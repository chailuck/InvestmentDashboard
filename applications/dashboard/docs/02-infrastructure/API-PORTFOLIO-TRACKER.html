API Spec — Portfolio Tracker

Objective: Serve portfolio positions, P&L calculations, and market index data from either the Excel-backed service or the PostgreSQL DB service, depending on the user's portfolio_mode setting (excel or db). The response shape is identical regardless of mode. Live prices are fetched from Yahoo Finance with a 5-minute in-memory cache. All endpoints require Bearer authentication.

Endpoints at a Glance

GET/api/v1/portfolio-tracker/raw-dataAll rows from the data source as-is (no P&L)

GET/api/v1/portfolio-tracker/market/global-indicesGlobal index prices (S&P 500, NASDAQ, BTC, Gold)

POST /api/v1/portfolio-tracker/refresh

Copies the read-only source Excel file to the writable working path, then writes a .cache_bust timestamp file so all Uvicorn workers drop their in-memory price caches. In DB mode, returns immediately with a message — no file is involved.

Response 200 — Excel mode

{
  "status":           "ok",
  "source":           "/app/investment_data/Investment tracking.xlsx",
  "destination":      "/app/uploads/portfolio.xlsx",
  "source_size_kb":   142.3,
  "destination_size_kb": 142.3,
  "message":          "File copied and cache cleared successfully."
}

Response 200 — DB mode

{
  "status":  "ok",
  "message": "Database mode — no file to refresh. Data is managed directly in the database."
}
503 Service Unavailable — source file does not exist at the configured path: {"detail": "Source file not found: /app/investment_data/..."}

GET /api/v1/portfolio-tracker/performance

Returns a time-series of realized (closed) P&L grouped by period. Used to draw the cumulative P&L line chart on the Portfolio page. Only closed positions are included.

Query parameters

ParamDefaultValues
perioddailydaily | weekly | monthly
from_date30 days agoISO date
to_datetodayISO date

Response 200

[
  {
    "date":          "2026-05",       // sort key: "YYYY-MM-DD" (daily), "YYYY-MM-DD" Mon (weekly), "YYYY-MM" (monthly)
    "label":         "May 2026",      // human-readable: "03/06/2026" | "02-06 Jun" | "May 2026"
    "dailyPnl":      8500.0,          // net P&L for this bucket
    "cumulativePnl": 24300.0          // running total from earliest to this bucket
  },
  {
    "date":          "2026-06",
    "label":         "Jun 2026",
    "dailyPnl":      3250.0,
    "cumulativePnl": 27550.0
  }
]

GET /api/v1/portfolio-tracker/performance/by-stock

Aggregates all positions (open + closed) by symbol. Used for the "by stock" breakdown table. Includes open positions using live prices for current value.

Response 200

[
  {
    "symbol":       "GULF",
    "net":          11750.0,      // total realized + unrealized P&L
    "investment":   57500.0,      // sum of entryPrice × positionSize
    "currentValue": 64000.0,
    "pnlPct":       20.43,
    "wins":         3,
    "losses":       1,
    "total":        4,
    "winRate":      75.0
  }
]

GET /api/v1/portfolio-tracker/market/set-indices

Fetches live Thai SET market index prices from Yahoo Finance. Note: the Thai SET Composite (^SETI) is unavailable on Yahoo Finance; SET50, SET100, and sSET are used as proxies. Results are cached for 5 minutes.

Response 200

[
  { "name": "SET50",  "value": 874.32, "change": -3.21, "changePct": -0.37 },
  { "name": "SET100", "value": 1923.55, "change": -5.10, "changePct": -0.26 },
  { "name": "sSET",   "value": 734.18, "change": 1.05,  "changePct": 0.14  }
]

Any field may be null if Yahoo Finance returns no data for that index.

GET /api/v1/portfolio-tracker/raw-data

Returns all rows from the data source without any P&L calculation. In Excel mode this is the raw spreadsheet content. Used by the "Raw Data" tab in the Portfolio page for debugging data quality.

Response 200

{
  "file":    "/app/uploads/portfolio.xlsx",   // or "database" in DB mode
  "columns": ["Symbol", "Entry Date", "Entry Price", "Exit Price", "Position Size", "..."],
  "rows":    [
    ["GULF", "2026-02-10", 57.5, null, 500, "..."],
    ["ADVANC", "2026-04-20", 215.0, 228.0, 200, "..."]
  ],
  "total":   2
}

Common Error Handling

StatusScenarioDetail
401Missing or invalid Bearer token"Not authenticated"
503Excel file not found (Excel mode)"Source file not found: /app/..."
500Pandas parsing error, yfinance timeout, or unexpected exceptionException message — check backend logs for stack trace
Yahoo Finance price data for Thai SET stocks is typically 15–20 minutes delayed during market hours. null price values indicate Yahoo Finance returned no data — the position's entry price is used as a fallback for open positions.
