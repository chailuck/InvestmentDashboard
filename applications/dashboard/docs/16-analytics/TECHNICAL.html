Query parameters

ParamTypeRequiredDefaultDescription

asset_typestringNoSETSET | CRYPTO | DR | US | HK | OTHER

Response — found

// GET /api/v1/analytics/search?q=PTT&asset_type=SET
{
  "symbol":     "PTT",
  "ticker":     "PTT.BK",          // actual Yahoo Finance ticker used
  "asset_type": "SET",
  "exchange":   "SET",
  "name":       "PTT Public Company Limited",
  "price":      37.25,             // last close price (4 decimal places)
  "change_pct": -0.67,             // % change vs previous close
  "found":      true
}

Response — not found

// GET /api/v1/analytics/search?q=INVALID&asset_type=SET
{
  "symbol": "INVALID",
  "found":  false
}
The search endpoint tries multiple ticker variants in order and returns the first that has data. For SET: SYMBOL.BK then bare SYMBOL. For CRYPTO: SYMBOL-USD, SYMBOL-USDT, then bare. For DR: SYMBOL.BK then bare. For HK: zero-padded 4-digit XXXX.HK, then SYMBOL.HK, then bare.

8. Indicator Calculations

8.1 RSI (Relative Strength Index) — 14-period

delta     = close.diff()
gain      = delta.clip(lower=0)
loss      = -delta.clip(upper=0)
avg_gain  = gain.ewm(com=13, min_periods=14).mean()     # EMA, not SMA
avg_loss  = loss.ewm(com=13, min_periods=14).mean()
RS        = avg_gain / avg_loss                          # 0 if avg_loss==0 → NaN guard
RSI       = 100 - (100 / (1 + RS))

8.2 Stochastic Oscillator — (9, 3, 3)

k_period  = 9   # lookback for highest high / lowest low
smooth_k  = 3   # smoothing period for raw %K
d_period  = 3   # smoothing period for %D (signal)

lowest_low    = low.rolling(k_period).min()
highest_high  = high.rolling(k_period).max()
raw_k         = 100 * (close - lowest_low) / (highest_high - lowest_low)
%K            = raw_k.rolling(smooth_k).mean()   # "slow %K"
%D            = %K.rolling(d_period).mean()       # signal line

8.3 VRVP (Volume Range Visible Profile) — 24 bins

price_min  = df["Low"].min()
price_max  = df["High"].max()
edges      = np.linspace(price_min, price_max, 25)   # 25 edges = 24 bins

for each candle:
    mid = (High + Low) / 2
    bin_idx = np.searchsorted(edges[1:], mid, side="left")  # which bin the candle's midpoint falls in
    volumes[bin_idx] += Volume

The resulting 24 {price_low, price_high, volume} records are sent to the frontend and rendered as a horizontal bar chart on the right Y-axis of the price pane.

9. Ticker Resolution by Asset Type

The backend tries each ticker in order, returning the first that has non-empty yfinance history.

asset_typeTicker candidates tried in order
SETSYMBOL.BK, then SYMBOL
CRYPTOSYMBOL-USD, SYMBOL-USDT, then SYMBOL
DRSYMBOL.BK, then SYMBOL
HK{SYMBOL zero-padded to 4 digits}.HK, SYMBOL.HK, SYMBOL
USSYMBOL (bare)
OTHERSYMBOL (bare)
All yfinance calls are executed in a thread pool via asyncio.get_running_loop().run_in_executor(None, ...) to avoid blocking the async event loop.

10. Security — Symbol Validation & Path Traversal Guard

Symbol whitelist regex

_SYMBOL_RE = re.compile(r"^[A-Z0-9.\-]{1,20}$")

Applied to every endpoint that accepts a symbol parameter. Symbols containing slashes, spaces, null bytes, or any other special character are rejected with HTTP 400 before any file system access.

_safe_glob — directory boundary enforcement

def _safe_glob(base_dir: str, pattern: str) -> list[str]:
    base = Path(base_dir).resolve()
    results = []
    for p in glob.glob(pattern):
        try:
            resolved = Path(p).resolve()
            resolved.relative_to(base)   # raises ValueError if outside base_dir
            results.append(p)
        except ValueError:
            log.warning("Path traversal attempt blocked", ...)
    return results

Even if a crafted symbol passed the regex, the _safe_glob function verifies each result is physically inside the configured directory (resolving symlinks). Any match outside the directory is silently dropped and logged as a warning.
